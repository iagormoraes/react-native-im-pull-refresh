# react-native-im-pull-refresh

A smooth pull to refresh component made with love.

## Installation

```sh
npm install react-native-im-pull-refresh
```

## Usage

```js
import { multiply } from "react-native-im-pull-refresh";

// ...

const result = await multiply(3, 7);
```
